About Playmat Studio

Playmat Studio is a free, browser-based image prep tool built for tabletop game players who want to print custom playmats with their own artwork. No installs, no subscriptions, no sign-up — just open the tool and start designing.

All image processing runs directly in your browser using the Canvas API and Fabric.js. Your artwork never leaves your machine, except for the optional AI upscaling and background-removal features, which use serverless Cloudflare Workers. Supports standard and extended mat sizes, plus specialty formats ranging from 10″×8″ up to 31″×12″.
